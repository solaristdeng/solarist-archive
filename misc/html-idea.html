内容优先：
一种静谧的秩序

你提到的“内容优先”，在设计上往往被误解为极简主义的空白。但真正的书籍设计告诉我们，空白不是“无”，而是结构的一部分。就像文艺复兴时期的排版师对待页边距那样，我们需要构建一种看不见的网格，让读者的视线在其中自然流淌。

几何与呼吸

当我们谈论网页设计时，往往忽略了“纸张”的物理属性。在这里，我们通过 CSS Grid 模拟了中国古籍中的版心与旁注关系。正文占据了主要的视觉焦点，而辅助信息则退居一侧，互不干扰。

这种布局允许视线在不需要跳转页面的情况下，获取多维度的信息。它是对称与不对称的博弈，也是秩序与自由的平衡。
